# 🗡️ Hầm Ngục Pixel - Pixel Dungeon RPG

## Game RPG Pixel 2D dò hầm ngục

### 🎮 Cách chơi

| Phím | Chức năng |
|------|-----------|
| `WASD` / `Phím mũi tên` | Di chuyển & Tấn công (đi vào quái) |
| `1-5` | Dùng bình thuốc trong túi |
| `Enter` | Bắt đầu / Chơi lại |

### 🎯 Mục tiêu

Chinh phục **10 tầng hầm ngục**! Mỗi tầng có quái vật mạnh hơn, vật phẩm quý hơn.

### ⚔️ Tính năng

- **4 loại quái vật**: Slime, Bộ Xương, Dơi Ma, Boss (mỗi 3 tầng)
- **Hệ thống chiến đấu**: ATK, DEF, chí mạng (15%), hụt (5%)
- **Lên cấp**: HP+15, ATK+2, DEF+1 mỗi cấp
- **Vật phẩm**: Kiếm, Bình máu, Chìa khóa mở cửa
- **Bẫy**: Cẩn thận bẫu trên đường đi!
- **Sương mù chiến tranh**: Chỉ thấy khu vực xung quanh
- **Minimap**: Góc phải trên cùng

### 🛠️ Công nghệ

- Next.js 16 + TypeScript
- HTML5 Canvas (pixel rendering)
- Tailwind CSS
- Không cần backend - chạy hoàn toàn trên trình duyệt

### 🚀 Chạy game

```bash
npm install
npm run dev
# Mở http://localhost:3000
```
