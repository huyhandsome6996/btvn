# 🗡️ Pixel Dungeon - Nhân Vật Chính

## Mô hình nhân vật chính cho game Pixel Hầm Ngục

### 📁 Danh sách file:

| File | Mô tả |
|------|--------|
| `pixel_hero_spritesheet.png` | Sprite sheet animation (idle, walk, attack) |
| `pixel_hero_portrait.png` | Chân dung nhân vật (màn hình chọn nhân vật) |
| `pixel_hero_modelsheet.png` | Model sheet nhiều góc nhìn (front, back, side, 3/4) |
| `pixel_hero_actions.png` | Các tư thế hành động (idle, chạy, chém, chắn, uống thuốc, niệm chú) |

### 🎮 Thông tin nhân vật:

- **Loại**: Hiệp sĩ dò hầm ngục (Dungeon Knight)
- **Trang bị**: Kiếm phép ánh sáng + Khiên sắt + Giáp da + Áo choàng đỏ
- **Phong cách**: Pixel art 16-bit, dark fantasy
- **Animation**: Idle, Walk, Run, Attack, Block, Drink Potion, Cast Spell

### 🛠️ Sử dụng:

Các asset này có thể dùng trực tiếp trong game engine:
- Unity / Godot / GameMaker Studio
- Import sebagai sprite sheet và cắt frame
- Kích thước pixel: 16x16 / 32x32 per frame
